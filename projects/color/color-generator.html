Build a Hex Color Generator

generate beautiful colors every time you press the spacebar

Getting Started

In this tutorial, we'll make our very own color generator with just a couple lines of JavaScript. First off, you'll need a folder with three files:

  • index.html - for our markup
  • style.css - for styling
  • app.js - for the function(s)

The !DOCTYPE html tag tells the web broswer version of HTML the page is written in. Right below that, we set some basic tags to specify the language. The head of a html document includes your title, meta tags, and your linked files. The viewport is used to make the page responsive so that any device can view the content clearly.
