Build a Text Editor

minimal editor that autosaves your text every second

Tweet

Getting Started

In this tutorial, we'll make our a text editor in the broswer. Using JavaScript, we can save the text automatically in a user's localStorage so that whenever the text editor is pulled up, it remembers the text that was written. localStorage is a JavaScript object that let's you save data in a browser.

As always, you'll need a folder with three files:

  • index.html - for our markup
  • style.css - for styling
  • app.js - for the function(s)

Since we're making a text-editor, our content has to be editable! Thankfully, HTML has an attribute for that. Go ahead: link the files, and create two divs with the contenteditable attribute and with the heading and content id's as seen below.

The reason we set div ids is that so that when we write our JavaScript, we can easily communicate with each of the divs. Now, if you open up your HTML file in the browser, you'll see that you have two editable text boxes. However, they're pretty ugly. Let's style them!

Styling the Editor

First off, we set our font to Helevetica, changed the color to make it more readable, and set the max-width of our text boxes to 50em. We also make our heading a bigger font and our content a little smaller. Now, look at your creation! You should have a beautiful, fully-functioning text editor, but if you refresh the page, does it save? Let's fix that.

JavaScript and localStorage function

To save our text in our two divs, we need to store it in the brower's localStorage. To do that, we first have to get the div ids from the HTML document. Then, we can set it to a localStorage instance with some default text. A function will check the div's content and save it in localStorage every second.

Once we set our two div's to localStorage['title'] and localStorage['text'] variables, we use the setInterval() function to save the innerHTML of each div every 1000 milliseconds, or 1 second.

There you have it! We now have built a text editor that autosaves. Try refreshing the page, or even better: exiting the browser! You should see your text still there! You can click here to view all the code for this project, and if you want to see what it should look like, you can view the demo :)

Try and changing up the style to your liking with CSS, or even add a footer! If you want a little challenge, try making your document title equal to your heading in the text editor. Happy Coding!
